# Valyrian Context for VS Code

> *"The Citadel (Git) tracks the history of the realm. ValyrianCtx tracks the memory of your intent."*

Valyrian Context is a CLI tool and VS Code extension for persistent AI coding context. It helps you maintain context across sessions, tools, and team members.

## Features

- **Auto-Capture**: Automatically captures context when you save files.
- **Save Context**: Manually save context with structured data (Task, Approaches, Decisions, State, Next Steps, Blockers).
- **Resume Context**: Resume context from previous sessions.
- **Log**: View session history.
- **Diff**: View changes since last save.

## Requirements

- Valyrian Context CLI installed globally or locally (`npm install -g valyrianctx`).
- Git initialized in the workspace.

## Extension Settings

This extension contributes the following settings:

* `valyrianctx.autoResume`: Enable/disable auto-resume on startup.
* `valyrianctx.defaultBranch`: Default branch to track context for.
